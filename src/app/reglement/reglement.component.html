sur la page des règlements et documents relatifs à la lutte contre le dopage. Cette section vous permet d'accéder aux textes officiels, aux guides et aux réglementations régissant les actions de lutte contre le dopage.

(01)

Code Mondial Antidopage (WADA)

Le Code mondial antidopage est la norme internationale régissant la lutte contre le dopage dans le monde entier, adoptée par l'Agence mondiale antidopage (WADA).

(02)

Code National Antidopage

Le Code national antidopage régit la législation locale concernant la lutte contre le dopage, incluant les sanctions et procédures juridiques spécifiques à notre pays.

(03)

Réglementations de l'Agence Nationale Antidopage (ANAD)

L'ANAD définit des régulations locales concernant les substances interdites, les méthodes de détection et les procédures à suivre pour assurer la conformité aux règles antidopage.

Liste des Substances Interdites

La liste des substances interdites mise à jour par la WADA comprend les substances et méthodes de dopage interdites en compétition et hors compétition.
